# 项目详情页 - 左右分栏设计

## 概述

将项目详情页的素材库/作品集 Tab 切换改为左右分栏布局，同时增强科技感视觉效果。

## 布局变更

| 项目 | 现状 | 变更后 |
|------|------|--------|
| 布局 | Tab 切换 | 左右分栏 |
| 比例 | - | 左侧 60% 作品集，右侧 40% 素材库 |
| 筛选 | 素材库有类型筛选 | 移除所有筛选按钮 |

## 科技感增强

### 背景层
- 保留现有动态模糊光晕
- 增加更多渐变层：青色、紫色、蓝绿色多层叠加
- 更通透的深色底

### 卡片层
- Grid 列数从 4 列增加到 5-6 列
- 增强玻璃拟态：半透明背景 + backdrop-blur
- 悬停时边缘发光效果
- 微妙的渐变边框

### 分割线
- 使用渐变色分割左右区域

## 实施步骤

1. 修改 ProjectDetail.tsx：移除 Tab 状态，改为左右分栏布局
2. 修改 MaterialsTab.tsx：移除 filter 状态和筛选按钮
3. 增强卡片样式：调整 grid、添加发光效果、渐变边框
4. 增加背景渐变层

## 预期效果

更酷炫、科技感更强的项目详情页，作品和素材一目了然。
