# 素材库设计文档

**日期**: 2026-03-17  
**状态**: 已批准

## 1. 需求概述

重构侧边栏菜单，新增「素材库」功能模块：
- 作品集：展示 AI 生成的结果卡片列表
- 工具箱：保持不变
- 素材库：用户手动上传的图片/视频管理，支持分组、标签、批量操作

## 2. 菜单结构调整

| 菜单项 | 路径 | 说明 |
|--------|------|------|
| 作品集 | /portfolio | 保持现有卡片列表，展示 AI 生成结果 |
| 工具箱 | /tools | 保持不变 |
| 素材库 | /materials | 新增，用户上传素材管理 |

## 3. 功能设计

### 3.1 支持的文件类型
- 图片：jpg, png, gif, webp
- 视频：mp4, webm

### 3.2 分组设计
- 纯标签分组（无文件夹层级）
- 每个素材可添加多个标签

### 3.3 操作方式
- **批量选择模式**：勾选多个素材后统一添加标签
- 点击上传按钮选择文件上传

### 3.4 展示样式
- 网格视图（卡片布局）

## 4. 数据模型

```typescript
interface Material {
  id: string
  type: 'image' | 'video'
  url: string
  filename: string
  size: number
  tags: string[]
  createdAt: string
}
```

## 6. 交互流程

1. **上传素材**
   - 点击「上传」按钮 → 选择文件 → 上传成功自动添加到列表

2. **批量添加标签**
   - 点击卡片选中（显示勾选状态）
   - 底部出现批量操作栏
   - 选择或输入标签 → 批量应用到所有选中项

3. **筛选**
   - 按标签筛选显示对应素材
   - 搜索框按文件名搜索
